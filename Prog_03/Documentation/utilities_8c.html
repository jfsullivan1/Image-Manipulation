returns a new output path with a .tga extension, built on a specified output directory provided by the user.

Parameters
paththis is the path of the input image, such as ../../Images/clown.tga
outputDirthis is the path of the output directory, provided by the user. Could be something like ../output
typethis is the type of operation that is going to be performed on the image. So, if we want to perform an operation such as mirrorV on the clown.tga file, we will pass "mirrorV" as an argument here. This is used for adding to the output string, so we can easily make clown [mirrorV].tga just by passing this argument, and won't have to have this function in each specific program, but instead can just have this one function and specify a type.
